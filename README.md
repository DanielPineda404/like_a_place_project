# 🌍 LikeAPlace – City Explorer

**LikeAPlace** es una aplicación web interactiva que permite explorar y gestionar lugares de interés dentro de una ciudad.

El usuario puede descubrir sitios según categorías como **gastronomía, cultura o deporte**, y organizar sus preferencias de forma **visual e intuitiva**.

---

## 📌 Descripción

LikeAPlace funciona como un **gestor de colecciones urbanas**, donde cada usuario puede:

* Explorar una ciudad *(inicialmente Medellín)*
* Navegar por categorías
* Descubrir lugares relevantes
* Guardar y clasificar sus preferencias

---

## 🎯 Objetivos

* Ofrecer una experiencia simple y visual para explorar la ciudad
* Permitir organizar lugares según intereses personales
* Construir una base para una app escalable de exploración urbana
* Aplicar y fortalecer habilidades en desarrollo web *(frontend → fullstack)*

---

## ✨ Funcionalidades

* 🌆 Exploración por ciudad
* 🧭 Navegación por categorías
* 🖼️ Visualización de lugares mediante cards
* 🔍 Vista detallada de cada lugar
